# Relay Logo Concepts (Minimal System)

This pack includes three minimal, modern SVG directions.

1. relay-logo-loop.svg
- Circular relay mark with a single directional cue.
- Best as the primary brand logo and website lockup.

2. relay-logo-beam.svg
- Minimal R monogram inside a rounded square.
- Best for app icon and favicon source.

3. relay-logo-nodes.svg
- Clean node-and-link symbol for workflow messaging.
- Best for diagrams, docs, and feature graphics.

## Recommendation

Use relay-logo-loop.svg as the default logo.

Use relay-logo-beam.svg for icons and compact placements.
